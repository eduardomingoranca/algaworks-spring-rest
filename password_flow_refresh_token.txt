<< Password Flow >>

Resource Owner informa as credencias para o Client
Client solicita o access token para o Authorization Server
Authorization Server retorna o access token para o Client

Access token expirou
Client faz a requisição em recurso da API usando access token para o Resource Server
Resource Server consulta access token o Authorization Server
Authorization Server retorna access token invalido para o Resource Server
Resource Server retorna nao autorizado para o Client

quando Access Token expira emite e utiliza o Refresh Token para gerar um Access Token
 

<< Password Flow Refresh Token >>

Resource Owner informa as credencias para o Client
Client solicita o access token para o Authorization Server
Authorization Server retorna o access token e refresh token para o Client

Access token expirou
Client solicita access token com refresh token Authorization Server 
Authorization Server retorna access token e refresh token Client
Client faz requisicao em recurso da API usando access token Resource Server
Resource Server consulta access token Authorization Server
Authorization Server access token invalido ao Resource Server
Resource Server retorna recurso solicitado ao Client 
Client usa recurso e mostra alguma coisa para o Resource Owner
