Nivel 00 - POX (Plain Old XML) 
HTTP apenas como mecanismo de transporte.
Este é o modelo RPC (Remote Procedure Call) ou seja
chamadas de procedimentos remotos. Possui um unico endpoint.

Nivel 01 - Recursos
Requisições para URIs com recursos.

Nivel 02 - Verbos HTTP
Introdução aos verbos HTTP de acordo com a semantica de cada verbo.

Nivel 03 - HATEOAS
HATEOAS (Hypertext As The Engine Of Application State) ou seja
Hipertexto como motor de estado da aplicacao. Este é o proximo
passo após realizar uma requisição, atraves de links.
