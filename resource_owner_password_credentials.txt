Fluxo Resource Owner Password Credentials.

Resource Owner informa as credenciais para o Client.
Client solicita access token para o Authorization Server.
Authorization Server retorna access token para o Client.
Client faz requisicao em recurso da API usando access token para o Resource Server.
Client consulta access token Authorization Server.
Authorization Server retorna access token valido para o Resource Server.
Resource Server retorna recurso solicitado para o Client.
Client usa recurso e mostra alguma coisa para o Resource Owner.
